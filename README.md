## Windows 安装方法

1. 点击工具对应的“下载最新版”。
2. 在 Release 的 Assets 中下载名称包含 `win.zip` 的文件。
3. 把 ZIP 完整解压到普通文件夹。
4. 双击文件夹里的 `启动工具.bat`。
5. 使用期间保持黑色启动窗口打开。

这些 Windows 包自带便携运行环境，不需要安装 Node、npm 或开发工具。

如果 Windows SmartScreen 弹出提示，请先确认文件来自本组织页并核对 SHA256，再选择“更多信息”。不要运行来自陌生转发地址的同名文件。

## 发布原则

- 每个工具使用独立仓库，避免版本和依赖互相污染。
- 新迭代使用新版本号，历史版本和下载包不覆盖。
- 玩家发行包优先提供离线 Windows ZIP；Android 支持以各工具说明为准。
- 工具仓库公开源码、构建方式、许可证和第三方声明。
